Editing Software

Templates should not be used with authoring software that amends HTML coding to conform to its own standards. Such software has the potential to alter the templates and render them unusable in some browsers. Netscape Composer and Claris Homepage are two such programs, while BBEdit and DreamWeaver are safe to use. Please check the manual of any authoring software before using it with the templates.

Generally text editors do not amend HTML, while most visual graphic editors do amend HTML to some extent. The only two visual graphic editors we are currently aware of that do not amend HTML are DreamWeaver and Go Live Cyber Studio. DreamWeaver is available from University Computer Supplies for both Macintosh and Windows.

Template Options

The templates below have been modified for conformance to the new Website look introduced 1 October 2001. Please also see information on modified navigational banners for use in conjunction with the new top blue bar.

Template 1.
This template is for Homepages and other high-level pages. It has a centrally positioned University crest and space for photographs on either side. This template is a text file and will need to be saved as an HTML file. Anyone who downloaded the HTML file here prior to 19 Nov should discard it and use the text file.

Template 2.
This template is for lower-level pages and has a reduced header. The University crest is positioned on the left hand side, there are no photographs and there is space in the header for the page title. As with the Template 1, it is a text file which you will need to save as an HTML file.

IMPORTANT
Please note that the footer (which is a server include) will only work on the main university server. If you are creating pages for use on another server, the include will have to be mirrored on that server — instructions are now available.
